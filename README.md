# SpaceBasic API

This repository contains automation scripts for interacting with the SpaceBasic API.

## Unofficial API

### Room and Bathroom Cleaning

`roomBathroomCleaning.py` - Automates the complaint workflow for room and bathroom cleaning requests.

**Features:**
- Fetches the latest complaint from your account
- Automatically adds a 5-star rating to the latest complaint
- Creates a new room/bathroom cleaning complaint with your preferred time slot

**Usage:**
1. Fill in your personal information in the configuration section
2. Run the script: `python roomBathroomCleaning.py`
3. Select an available time slot when prompted
